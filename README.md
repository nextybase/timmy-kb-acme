# timmy-kb-template

Template di base per la costruzione della Knowledge Base “Timmy” destinata ai nuovi clienti Nexty.

## Cosa contiene?

- **Struttura di cartelle vuote** (`output/`, `src/`, `config/`), pronte per essere popolate durante l’onboarding.
- File `.gitkeep` per mantenere la struttura su Git.
- README e istruzioni base.

## Come usarlo

1. **Clona questo repo** per ogni nuovo cliente Nexty.
2. Rinomina la repo secondo lo standard:  
   `timmy-kb-nomeazienda`
3. Segui la guida di onboarding per configurare:
    - Cartella Drive condivisa con il cliente (input dati)
    - File di configurazione YAML nella cartella `config/`
4. Esegui la pipeline di ingestione e pubblicazione Markdown.
5. Collega il repo al GitBook dedicato del cliente, se necessario.

---

> **NB:** Questo template NON contiene dati sensibili né knowledge base. Serve solo per avviare la struttura standardizzata di Timmy!
